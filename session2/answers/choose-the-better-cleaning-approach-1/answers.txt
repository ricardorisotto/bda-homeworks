Quiz: choose the better cleaning approach

Essay Question:
Assume that `my_data` is a list with many rows, and some values are missing (represented as empty strings "").

You can:
1. Clean in place (modify the original list directly), or
2. Copy first to a new list, then clean the copied list.

Compare the two approaches in terms of:
- time complexity
- space complexity
- risk/safety

Then explain which option you would choose for:
- a quick one-off local script
- a production workflow where raw data must be preserved

Be specific with Big-O notation (for example `O(n)`, `O(1)`, `O(n)` extra space).

Student Answer:
A list with many rows would be a n x r matrix, where m is the number of rows and r is the number of elements in each row. So option 1, to clean in place would take O(n x r) time complexity, but would be O(1) space complexity. Option 2, to copy to a new list would be one operation to copy, so time complexity would still be O(n x r) to clean the new list. But the space complexity would now be O(n x r) because of the copy. Even so, having a copy is much safer and less risky as you will not accidentally delete data that is then unrecoverable.
