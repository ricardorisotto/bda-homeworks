Quiz: Complexity Basics

Essay Question:
Describe the `time` complexity of:
A) searching a number in an unsorted list
B) searching a number in a sorted list using binary search
C) sorting a list using bubble sort

Provide a brief explanation for each case (for example, justify why the complexity is O(n)).

Student Answer:
A) An unsorted list means we have to search the whole list to find the number. So time complexity is O(n) where n is the length of the list.
B) Binary search means picking a number in the middle of the sorted list. If the number picked is greater than what we search for then we search in the part of the list containing numbers less than that number, otherwise we search in the other part. We continue picking the middle of each part and splitting the parts recursively until we find the number. The time complexity of such a search is O(logn). 
C) using bubble sort to sort a list in the worst case scenario means comparing each element in the list with every other elemenet in the list, since bubble sort works by comparing numbers adjacent and swapping if one is greater than the other. So that makes O(n^2) i.e. O(n squared).
